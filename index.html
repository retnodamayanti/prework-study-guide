Prework Study Guide

HTML

  • The head element contains information or metadata and it's not visible to user
  • The body element represents visible content to the user. All element that we want to show to the user must be placed here.
  • Elements provide framework and structure of the webpage/HTML.

CSS

  • External CSS Style Sheet: Developers keep all their CSS rules in a separate file link using <link>

Git

  • Every project on GitHub is called Repositery (Repo).
  • Branch is a workspace where you can make changes that don't affect the live site.

JavaScript

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout: switch to another branch
  • git add -A: commit all files in the working branch
